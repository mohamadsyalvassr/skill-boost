# Apa itu Cloud ?

Bayangkan kamu sedang menyalakan lampu di rumah. Kamu tidak perlu punya pembangkit listrik sendiri—cukup sambungkan ke PLN dan bayar sesuai pemakaian. Nah, **Cloud Computing** itu **mirip seperti langganan listrik**, tapi untuk layanan digital seperti server, penyimpanan data, dan aplikasi.

⚡ **Analogi Utama:** Cloud adalah layanan digital yang bisa Kamu pakai _tanpa harus punya perangkat sendiri_—persis seperti:

* **Listrik dari PLN:** Kamu pakai tenaga listrik tanpa punya genset.
* **Netflix:** Anda bisa nonton film tanpa perlu koleksi kaset atau punya server film di rumah.

➡️ Jadi, alih-alih beli dan kelola server fisik sendiri (yang mahal dan rumit), Kamu tinggal menyewa dari penyedia Cloud seperti AWS, Azure, atau Google Cloud. Anda bayar sesuai penggunaan, dan bisa langsung dipakai.
